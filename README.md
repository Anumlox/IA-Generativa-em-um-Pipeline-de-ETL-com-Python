# IA-Generativa-em-um-Pipeline-de-ETL-com-Python

## Descrição 
- O uso de uma IA Generativa em um Pipeline de ETL com Python onde essa IA ajudaria pessoas que estariam passando por momentos difíceis e que estariam precisando de algum conselho ou uma ajuda.
---------------

## Exemplos
- Mensagem: "Estou ansioso demais ultimamente" - Aqui teriamos a mensagem recebida pela IA.
- Resposta: "Sinto muito que você esteja passando por isso… ansiedade pode ser bem pesada. Tente fazer uma pausa, respirar fundo e focar em algo que te acalme por alguns minutos. Se possível, procure alguém de confiança para conversar — compartilhar alivia bastante." - Essa seria a resposta de apoio.
